## Development

**Do Not Push to the `master` branch at once.**

1. Create a new branch for development

```bash
git checkout -b dev/{{yourname}}
```

2. Stage the files

```bash
git add .
```

3. Commit the files

```bash
git commit -m "{{message}}"
```

4. Push the update

```bash
git push origin dev/{{yourname}}
```
